# nifty50pred
It's an decision tree and random forest algorithm that is fed on almost 35 years of data of nifty 50 and now it predicts the movement of it(up/down) based on several factors including RSI, MA, Volatility. THIS IS ONLY FOR EDUCATIONAL PURPOSE. PLEASE DON'T BLINDLY TRUST THIS MODEL AND  MAKE DECISIONS COMPLETELY BASED  ON MODEL.
